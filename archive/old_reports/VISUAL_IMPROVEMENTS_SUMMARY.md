## ✨ 任务3：进度显示优化

### 改进内容

#### 1. Rich 进度条优化 (`xet/cli/progress.py`)

**改进前**：
```python
TextColumn("[bold blue]{task.description}"),
BarColumn(bar_width=40),
```

**改进后**：
```python
TextColumn("[bold cyan]{task.description}", justify="left"),
BarColumn(bar_width=40, style="cyan", complete_style="green"),
transient=False,  # 完成后保留进度条
```

**效果**：
- 🎨 使用青色主题，完成部分显示绿色
- 📌 进度条完成后保留，方便查看历史
- 🔤 文本左对齐，更整洁

#### 2. 简单进度条优化

**改进前**：
```
Downloading: 75.0% [=======================>      ] 75.0 MB/100.0 MB  5.0 MB/s  ETA: 5s
```

**改进后**：
```
📥 75.0% [████████████████████████████████░░░░░░░░] 75.0 MB/100.0 MB │ ⚡ 5.0 MB/s │ ⏱ 5s │ 📦 8/10 │ 🧩 3/5
```

**改进点**：
- 🎯 使用 Unicode 实心/空心块（█/░）代替 ASCII 字符
- 📊 emoji 图标提升可读性
- 📦 显示 Xorb 进度
- 🧩 显示 Segment 进度（分段下载时）
- 🔗 使用竖线分隔符（│）更清晰

#### 3. info 命令输出美化

**改进前**（纯文本）：

**改进后**（使用 Rich Panel + Table）：

**改进点**：
- 📦 使用 Rich Panel 框架包裹
- 📊 表格布局更整齐
- 🎨 颜色高亮关键信息
- 🔤 emoji 图标增强可读性
